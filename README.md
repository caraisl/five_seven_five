## Packages, Libraries and other sources used
This project makes use of Django, HTML, CSS, javascript, and AJAX

[Google fonts](https://fonts.google.com/) was used for websites fonts

A [Font Awesome](https://fontawesome.com/) icon kit was used for the comment, like and post icons throughout the site. **NOTE: When a page is loaded, this kit creates an inline stylesheet for these icons. This stylesheet is not created by us and is not in the source HTML**

To validate the number of syllables in each haiku, the [cmudict](https://pypi.org/project/cmudict/) python library was used. The [syllables](https://pypi.org/project/syllables/) python library was used to estimate the number of syllables in a word if it was not in the dictionary.

The background image for the site is [Hokusai's "The Great Wave off Kanagawa"](https://www.metmuseum.org/art/collection/search/45434), which is in the public domain.
